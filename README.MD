# Essentia
[![Build Status](https://travis-ci.com/essentiaone/Essentia-iOS.svg?token=KxqSd9shNNgW2Ln2sojL&branch=develop)](https://travis-ci.com/essentiaone/Essentia-iOS)

### Usage
```
cd Essentia
pod install
pod repo update
pod update
open Essentia.xcworkspace
```
### Requirements
- `xCode 10.0`
- `iOS 11.0`
- `Swift 4.2`
