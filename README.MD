Essentia One iOS application. Manage your crypto assets in one app.

## How to setup Essentia app
```
cd Essentia-iOS
pod install
open Essentia.xcworkspace
```
## Requirements
- `xCode 10.0`
- `iOS 11.0`
- `Swift 4.2`
